  2. 在编译前先清空一次目录,sudo make distclean

  3. 修改编译配置,make my2440_config

  4. make

  5. sudo make

  6. 不出意外的话，就能顺利通过，如果没有通过的话，我们可以看看根目录下的.config文件的配置是够正确
